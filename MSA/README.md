# Linear Probing Models

## Usage
(1) training
```
$ cd training
(8-fold validation)
$ ./EXE-HARMONIX-8fold.sh
(cross-dataset validation)
$ ./EXE-CROSS.sh
$ cd -
```

(2) evaluation
```
$ cd evaluation
(8-fold validation)
$ ./EXE-HARMONIX-8fold.sh
(cross-dataset validation)
$ ./EXE-CROSS.sh
$ cd -
```
